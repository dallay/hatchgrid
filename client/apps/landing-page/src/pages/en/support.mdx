# Support Hatchgrid

Thank you for your interest in supporting Hatchgrid! As an open-source project, we rely on community support to continue developing and improving our platform.

## Ways to Contribute

### 1. Code Contributions

If you're a developer, there are many ways to contribute:

- Fix bugs and issues in our [GitHub repository](https://github.com/hatchgrid/hatchgrid)
- Implement new features
- Improve documentation
- Write tests to ensure stability

Our codebase is built with Kotlin and Spring Boot on the backend, with a modern TypeScript and React frontend. Check our [contribution guidelines](https://github.com/hatchgrid/hatchgrid/blob/main/CONTRIBUTING.md) to get started.

### 2. Documentation

Good documentation is crucial for any project. You can help by:

- Improving existing documentation
- Creating tutorials and guides
- Translating documentation to other languages
- Creating video tutorials

### 3. Community Support

- Answer questions in our [community forums](https://community.hatchgrid.com)
- Help triage issues on GitHub
- Share your experiences and use cases

### 4. Financial Support

Your financial contributions help us dedicate more resources to development:

- [GitHub Sponsors](https://github.com/sponsors/hatchgrid)
- [Open Collective](https://opencollective.com/hatchgrid)

## Getting Started

To start contributing, please join our [Discord server](https://discord.gg/hatchgrid) where our team and community members can help guide you.

Thank you for supporting Hatchgrid and helping us build a better platform for everyone!
